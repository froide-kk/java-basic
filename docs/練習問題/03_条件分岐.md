## 条件分岐

乱数の作り方

```java
import java.util.Random;
class Sample {
  public static void main(String args[]) {
    Random rand = new Random();
    int num = rand.nextInt(10);
    System.out.println(num);
  }
}
```

`nextInt()` は 0 から引数に指定した値未満の整数を返します。

`nextInt(10)` では 0 から 9までの整数が取得できます。
